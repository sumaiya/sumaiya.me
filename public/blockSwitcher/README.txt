Block Switcher: 
A simple game using Linked Lists. 
Rearrange the blocks until you reach the target, in as few moves as possible.

To run from command line:
1) make sure all java files are compiled
2) navigate to folder containing Switcher.html
3) type appletviewer Switcher.html
4) select a difficulty and press Start to play!
